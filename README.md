# REST-API
REST-API in golang for CRUD'ing movies in mongoDB. 

# Requirements
- Golang
- Mongo DB
- Postman to test the API's
- Set configurations in config.toml file

# Run
Edit config file to set sever port and database. 
Then in the project directory run

`$ go run main.go`  

# Running Tests
`$ go test ./tests/*`
